# Smiles School Webpage

## Overview
This repository contains the HTML structure for the Smiles School landing page. The page showcases tutorials, instructors, popular content, membership details, and frequently asked questions (F.A.Q.), all designed with accessibility and responsiveness in mind.

---

## File Structure

---

## Features

### Header & Navigation
- The page includes a `<header>` with a `<nav>` containing:
  - Logo (displayed via an image)
  - Navigation links: **COURSES**, **PRICING**, and **LOGIN**
- Responsive layout using CSS flexbox for horizontal alignment.

### Main Content Sections
1. **Hero Section**
   - Headline: "Get Schooled"
   - Highlighted features: "SMILES", "GRIN", "LAUGH"
   - Register button: "REGISTER FOR FREE"

2. **Instructors Section**
   - Highlights instructors with profile images, names, and achievements.
   - Structured using `<section>` and `<div>` elements for each instructor.

3. **Testimonials Section**
   - Showcases student testimonials with images and blockquotes.

4. **Popular Tutorials Section**
   - Displays tutorials with:
     - Thumbnail images
     - Titles
     - Descriptions
     - Author information
     - Duration and ratings

5. **Membership Section**
   - Promotes free membership with four highlight items.
   - Includes a call-to-action button.

6. **F.A.Q. Section**
   - Frequently asked questions arranged in multiple columns.

---

### Footer
- Contains the site logo and social media links (Facebook, Twitter, Instagram) using **Font Awesome** icons.
